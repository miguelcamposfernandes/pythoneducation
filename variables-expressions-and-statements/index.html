1.2 Variables

One of the most powerful features of a programming language is the ability to manipulate variables.  A variable is a name that refers to a certain value. For example:
message = “Hi! How are you?”
x = 12

This example makes 2 assignments. The first one assigns a string to a new variable called message and the second one assigns an integer value to x.

1.3 Operators and operands

Operators are special symbols that represent computations like addition and multiplication. The values the operator is applied to are called operands.
The operators +, -, *, /, and ** perform addition, subtraction, multiplication,
division, and exponentiation, as in the following examples:

2 + 2
x - 42
Hour / 60
Minute * 60
3 ** 3

1.4 Modulus Operator

The modulus operator works on integers and yields the remainder when the first operand is divided by the second. In Python, the modulus operator is a percent sign (%). The syntax is the same as for other operators:

remainder = 9 % 4
print(remainder)

>> 1

After running the program above you should notice that the output is the number ‘1’. This basically means that ‘1’ is the remainder of 9/4.

The modulus operator turns out to be surprisingly useful. For example, you can check whether one number is divisible by another: if x % y is zero, then x is divisible by y.

1.6 Comments

As apps get bigger and bigger, they get far more difficult to read and understand. So we’ll need to use comments in order to explain and sort our code. I’ve been using comments all the time through out this chapter to explain to you the code I was writing.

To use comments in Python you’ll need to add an ‘#’. Like this:

# Asking the user for his name
name = input("What is your name: ")
# Greeting the user with his input
greeting_msg = print("Hi " + name + "! How are you?")

Everything from the ‘#’ to the end of the line will be ignored. And has no effect on the program whatsoever.
