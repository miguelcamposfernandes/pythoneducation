4.1 Updating variables

In programming you will often have the need to update a variable, where the new value of the variable depends on the old.
 
x = x + 1

This means “get the current value of x, add 1, and then update x with the new value.”

You can not update a variable if you haven’t created it before. So, before can update it you’ll have initialize it, using a simple assignment.

x = 9
x = x + 1

4.2 The while statement

One thing computers can do very well and humans can’t are repetitive tasks without errors. And one of others forms of iteration is the while loop.

This type of statement is also referred as a loop, because after subtracting 1 from i the program starts from the top again until the condition becomes false. Each time we execute the body of the loop an iteration. So we would say that this program has 10 iterations.
