3.1 Function calls

In programming languages, a function is a sequence of statements that performs a computation. When you define a function you can give it a name that will be then used to call the function. An example of the function would be the type() function that we’ve already covered:
 
>>> type(PythonEducation.org)
<class 'str'>

The name of the function is type. And the expression inside the parentheses is the input we’re passing the function also called argument. The function then “takes” the argument and “returns” a result, in this case whether it is a string, an integer or a float.
