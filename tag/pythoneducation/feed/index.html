5.1 A string is a sequence

A string is nothing more than a sequence of character, which can be accessed one at the time:
 
fruit = "banana"
letter = fruit[1]

Using the brackets expression the second statement extracts the character at index position 1 and assigns into the variable letter.

But you might not get what you expect:

>>> print(letter)
a

You probably expected the letter “b” as it is the first letter in the word banana. But Python works with index’s and the first character is the 0th position and the second in the 1th position. 

  

5.2 Getting the length of a string using len

len is one of many built-in functions Python has, which returns the number of characters in a string:
 
>>> fruit = "banana"
>>> len(fruit)
6

6.1 A list is a sequence

Like a string, a list a sequence. A string is a sequence of characters and a list is a sequence of any type (Integers, Strings, Floats, etc.). The values inside a list are called elements or items.
A list can be created by enclose the items in square brackets:
 
[1, 2, 3, 4, 5]
["Chandler", "Phoebe", "Ross", "Rachel", "Monica", "Joey"]

6.2 Lists are mutable

If you want to access a specific item inside a list you use the same procedure as for accessing the character in a string: the bracket operator. The expression inside the brackets specifies the index:
 
>>> friends = ["Chandler", "Phoebe", "Ross", "Rachel", "Monica", "Joey"]
>>> print(friends[0])
Chandler

The in also works in lists:

>>> "Chandler" in friends
True
>>> "Janice" in friends
False

6.3 List operations

The + merges lists together:
 
>>> x = [1, 2, 3]
>>> y = [4, 5, 6]
>>> merge = x + y
>>> print(merge)
[1, 2, 3, 4, 5, 6]

The *  repeats a list given a number of times:

>>> ["PythonEducation.org"] * 3
["PythonEducation.org", "PythonEducation.org", "PythonEducation.org"]
>>> [0, 0, 7] * 3
[0, 0, 7, 0, 0, 7, 0, 0, 7]

6.4 List slices

You can also you the slice operator on lists:
 
>>> friends = ["Chandler", "Phoebe", "Ross", "Rachel", "Monica", "Joey"]
>>> friends[1:3]
["Phoebe", "Ross"]
>>> friends[:4]
["Chandler", "Phoebe", "Ross", "Rachel"]
>>> friends[3:]
["Rachel", "Monica", "Joey"]
