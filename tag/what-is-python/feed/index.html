What is Python?

Python is an interpreted, object-oriented, high-level programming language created by Guido van Rossum in 1991.

But… Why Python?

What makes Python different from every other programming language out there? Well, the main reasons are that:

  • Python is very easy to learn and use;
  • Mature and Supportive Community;
  • 100’s of Libraries and Frameworks;
  • Versatility, Efficiency, Reliability and Speed;
  • Big data, Machine Learning and Cloud Computing

Python in Real-World Scenarios:

  • Web Deveploment
  • Machine Learning 
  • Data Science 
  • Computer Science Education
  • Computer Vision and Image Processing
  • Game Deveploment
  • Medicine and Pharmacology
  • Biology and Bioinformatics
  • Neuroscience and Psychology
  • Astronomy
  • And a bunch of other areas such as business, robotics, meterology and graphical user interface (GUI) development.
