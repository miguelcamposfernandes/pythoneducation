7.1 Getting started with dictionaries

A dictionary is like a list, but in a list the index positions must be integers; in dictionaries it can be almost any type.
You could think of a dictionary as a mapping between a set of indices, also known as keys and a set of values. In the example we’re going to do, we’ll build a dictionary that maps from Portuguese to English words, so the keys would be the Portuguese words and the values the English ones.
To first create a dictionary we should begin by creating a variable and assign the built-in function dict():
 
>>> pt2eng = dict()
>>> print(pt2eng)
{}

Now, we successfully created a dictionary, but it’s empty. To add items, you can use square brackets:

>>> pt2eng["um"] = "one"

In this case the key value would be the Portuguese word “um”  and its value the English word “one“. By printing it out, we can see the key-value pair with a colon in between:

>>> print(pt2eng)
{"um": "one"}

You can also create a dictionary with more than 1 key-value pair:

>>> pt2eng = {"um": "one", "dois": "two", "tres": "three"}
>>> print(pt2eng)
{'um': 'one', 'dois': 'two', 'tres': 'three'}

You can now use the keys to look up the corresponding values:

>>> print(pt2eng["tres"])
"three"

The len function also works with dictionaries, it returns the number of key-value pairs:

>>> len(pt2eng)
3

The in operator works in dictionaries too, it tells you whether something appears as a key in the dictionary, a value will return False.

>>> "um" in pt2eng
True
>>> "one" in pt2eng
False
